Node version used: v22.14.0
npm version: 10.9.2
Libraries used: React and Vite

Steps to run project:
1. Open new terminal
2. Write: npm install
3. Run with: npm run dev -> port is picked for use.
4. Write in browser http://localhost:<port>
5. Enjoy the software
